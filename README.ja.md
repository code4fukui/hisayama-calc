# hisayama-calc

久山町の心臓病リスク計算アプリです。性別、収縮期血圧、糖代謝異常、LDL-コレステロール、HDL-コレステロール、喫煙状況を入力すると、年代別の心臓病リスクを計算して表示します。

## デモ
[https://code4fukui.github.io/hisayama-calc/](https://code4fukui.github.io/hisayama-calc/)

## 機能
- 性別、収縮期血圧、糖代謝異常、LDL-コレステロール、HDL-コレステロール、喫煙状況を入力
- 心臓病のリスクポイントを計算
- 年代別の心臓病リスクを表示

## 使い方
ブラウザで上記のデモサイトにアクセスし、各項目を入力してください。自動的にリスクが計算されて表示されます。
