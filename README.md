# hisayama-calc

A calculator for estimating cardiovascular disease risk in Hisayama, Japan.

## Demo
[https://code4fukui.github.io/hisayama-calc/](https://code4fukui.github.io/hisayama-calc/)

## Features
- Calculates risk of cardiovascular disease based on user input
- Displays risk percentages for age ranges 40-49, 50-59, 60-69, and 70-79
- Colorizes risk levels (low, medium, high)

## Usage
The application can be used directly in the web browser at the demo link above.
